# Projeto Ecommerce

## Descrição
Este é um projeto de um banco de dados para um sistema de ecommerce, com tabelas para clientes, produtos, pedidos, fornecedores e mais.

## Estrutura do Banco de Dados
O banco de dados é composto pelas seguintes tabelas:
- `client`: Contém informações dos clientes.
- `product`: Contém informações sobre os produtos.
- `orders`: Contém informações sobre os pedidos feitos pelos clientes.
- `seller`: Contém informações sobre os vendedores.

## Como rodar
1. Importe o script SQL no seu banco de dados.
2. Execute as consultas para testar o funcionamento do sistema.
